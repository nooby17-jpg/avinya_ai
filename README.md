# AVYA – AI Hospital Receptionist

AVYA is an AI-powered Interactive Hospital Receptionist and Emergency Assistance System Using Conversational Intelligence

## Features
- ML & Deep Learning (LSTM) based emergency detection
- Rule-based safety override (ethical AI)
- FastAPI backend
- Hospital-specific configuration
- Designed for kiosk, web, and mobile deployment

## Tech Stack
- Python
- FastAPI
- Scikit-learn
- TensorFlow (LSTM)
- JSON-based hospital configuration
